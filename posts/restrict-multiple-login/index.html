Concurrency Login / Restrict Multiple Login

Sep 30, 2020
1 min read

Problem Description

  • Same user can login from another PC concurrently.
  • It can lead to data compromised by attacker.
  • Attacker can take control of this system.

Solution

  1. Check, whether requested user is already logged in.
  2. Do not share user information with others.
  3. Two-factor authentication could be implemented to be more secured.
